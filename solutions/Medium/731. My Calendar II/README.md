# 731. My Calendar II

## Solution1

Traversal + brute force

Time complexity of `book()` method: `O(n)`

Space complexity: `O(n)`

## Solution2

Difference array

Time complexity of `book()` method: `O(n)`

Space complexity: `O(n)`

## Solution3

Segment tree

Time complexity of `book()` method: `O(log C)` where `C = 10 ^ 9`

Space complexity: `O(n log C)`
